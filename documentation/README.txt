Drawing UML with PlantUML
=========================

Making UML diagrams is very easy with PlantUML.
Use documentation for PlantUML on your language: http://plantuml.com/ru/guide
You can create or modify diagrams as text files _without_ installation of anything!
These steps below are to convert your text files into beautiful images.

Installation
------------

You need to:

1. download last version of PlantUML here:
   http://plantuml.com/ru/download
2. Install simpliest JRE environment:
   # apt-get install openjdk-11-jre-headless

Running
-------

Enter to the documentation directory locally and do
# java -jar plantuml.jar classes_diagram.txt
